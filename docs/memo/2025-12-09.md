- 修复 SQL Agent 生成不完整 SQL 导致的 MySQL 语法错误问题：
  - 在 agent-langchain/agent_langchain/db/sql_agent.py 的 _ensure_safe_select_sql 中增加结构性校验：
    - 对以 SELECT 开头的语句要求包含 FROM 子句，避免类似裸 \"SELECT\" 被直接发送到数据库。
