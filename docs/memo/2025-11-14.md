- 结论：当前 GPU 零拷贝分支因为 NV12→NCHW 预处理使用最近邻插值，相比 CPU 双线性路径会在小目标和边缘区域产生信息损失，表现为“检测框不如 CPU 路径准确”。后续可通过为 CUDA kernel 增加双线性分支，并默认对检测模型使用双线性插值来收敛两条路径的精度差异。
