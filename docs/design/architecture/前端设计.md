懂你说的痛点：默认堆出来的 Element Plus UI 很“企业内网风”，离专业的视频监控/视频分析系统还差一截。下面给你一套**可直接替换的设计体系 + 主题 + 组件用法规范 + 关键页面代码骨架**，目标是：**工业级深色主题、专业信息密度、清晰的层次和状态语义**。你把这些落到现有项目里，观感会立刻“提质提色”。

------

# 0. 总体风格与目标

- **基调**：工业深色（暗蓝/石墨），高对比但不刺眼；卡片层次明显，视频内容“主角化”。
- **信息密度**：表格紧凑、小尺寸控件；仪表指标清晰读数；状态“绿跑、黄告、红错”一眼明了。
- **交互**：轻动效（悬停/聚焦），操作有反馈（加载、禁用、回滚）。
- **元素**：方圆 8px；阴影收敛；图标统一（Element Plus 图标）。

# 2. Element Plus 工程化与密度规范

**Vite 插件**：按需自动引入，控件全局走 **small** 尺寸、圆角 8px。
 `vite.config.ts`（关键片段）

```
import { defineConfig } from 'vite'
import vue from '@vitejs/plugin-vue'
import Components from 'unplugin-vue-components/vite'
import AutoImport from 'unplugin-auto-import/vite'
import { ElementPlusResolver } from 'unplugin-vue-components/resolvers'

export default defineConfig({
  plugins: [
    vue(),
    AutoImport({ resolvers: [ElementPlusResolver()], dts: true }),
    Components({ resolvers: [ElementPlusResolver()], dts: true })
  ]
})
src/main.ts
import { createApp } from 'vue'
import App from './App.vue'
import router from './router'
import { createPinia } from 'pinia'
import './styles/theme.css'               // ← 引入主题
import 'element-plus/theme-chalk/dark/css-vars.css' // 兼容 Element Dark Var

const app = createApp(App)
app.use(router).use(createPinia())
app.mount('#app')
```
